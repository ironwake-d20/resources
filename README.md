# Ironwake Resources

A collection of tools for the Ironwake campaign, built as a React web app.

## Tools

- **Dice Roller** — Roll and keep dice for any purpose.

## Tech Stack

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/) with the [Catppuccin](https://github.com/catppuccin/tailwindcss) theme
- [React Router](https://reactrouter.com/)

## Development

```bash
pnpm install
pnpm dev
```

Other scripts:

```bash
pnpm build        # production build
pnpm preview      # preview production build
pnpm typecheck    # type-check without emitting
pnpm format       # format source files with Prettier
```
